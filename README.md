# Ford Smart Analytics 🚘

Aplicativo mobile desenvolvido em React Native + Expo para o Challenge Ford FIAP.

O sistema simula uma plataforma inteligente de análise automotiva, permitindo consultas de veículos Ford, gerenciamento de favoritos, histórico de pesquisas e dashboard executivo com métricas inteligentes de retenção e pós-venda.

---

# 📱 Funcionalidades

## 🔍 Consulta Inteligente
- Busca de veículos por:
  - Marca
  - Modelo
  - Versão

- Exibição de:
  - Ano
  - Categoria
  - Preço
  - Especificações técnicas
  - Score de retenção
  - Insights de IA

---

## ⭐ Favoritos
- Salvar veículos favoritos
- Remover favoritos
- Persistência local com AsyncStorage

---

## 🕓 Histórico
- Registro automático das consultas realizadas
- Exibição de data e hora
- Limpeza completa do histórico

---

## 📊 Dashboard Executivo
- Total de consultas
- Total de favoritos
- Marca mais pesquisada
- Quantidade de veículos cadastrados
- Preço médio da base
- Ranking de veículos
- Status operacional da IA
